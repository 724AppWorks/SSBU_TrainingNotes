# 概要
キャラクターについてメモ(動かし方、コンボ、対策など）

# ディレクトリ構成
探しやすくしておきたいので以下の階層でいきます

## character name (use)
- 使用する側のキャラクター名
- 基本的に以下の項目を書く
  - 概要
  - 立ち回り
  - コンボ

### matchup
キャラ別対策資料置き場
#### character name (opponent)
対戦相手のキャラ別対策

### report
- オンラインなどで保存した動画置き場
- 動画URLと一緒に反省内容を書く
